Welcome to **7POINTS.NET (Pisukide)**! This quickstart guide will help you go from zero to managing your first shared apartment in minutes.

## What You'll Accomplish

By the end of this guide, you will:
- Create your user account
- Create or join an apartment
- Add your first expense
- View your personalized dashboard
- Understand the core features

<Note>
  7POINTS.NET is designed specifically for university students sharing apartments. It helps manage household tasks, expenses, and coordination effortlessly.
</Note>

## Prerequisites

Before starting, make sure you have:
- A valid email address
- Access to the 7POINTS.NET web application
- (Optional) An apartment code if joining an existing apartment

### Step 2: Create or Join an Apartment

You have two options:

#### Option A: Create a New Apartment

If you're the first person setting up your shared apartment:

1. Click **"Sortu Pisua"** (Create Apartment)
2. Enter your apartment name (up to 25 characters)
3. Click **"Gorde Odoon"** (Save to Odoo)

The system automatically generates a **5-digit code** that your roommates can use to join.

```php Create Apartment Backend
// From: app/Http/Controllers/PisoController.php:82
public function store(Request $request)
{
    $validate = $request->validate([
        'pisuaren_izena' => 'required|string|max:255',
    ]);

    $user = $request->user();

    // Generate unique 5-digit code
    $kodigoa = null;
    do {
        $posibleKodigoa = str_pad(rand(0, 99999), 5, '0', STR_PAD_LEFT);
        if (!Piso::where('kodigoa', $posibleKodigoa)->exists()) {
            $kodigoa = $posibleKodigoa;
        }
    } while (!$kodigoa);

    $pisua = Piso::create([
        'izena' => $validate['pisuaren_izena'],
        'kodigoa' => $kodigoa,
        'odoo_id' => null,
        'user_id' => $user->id,
    ]);

    // Attach user as coordinator
    $pisua->inquilinos()->attach($user->id, ['mota' => 'koordinatzailea']);

    // Sync to Odoo
    Bus::chain([
        new SyncUserToOdoo($user),
        new SyncPisoToOdoo($pisua),
        new SyncInquilinoToOdoo($user, $pisua)
    ])->dispatch();

    session(['pisua_id' => $pisua->id]);
    return redirect()->route('dashboard');
}
```
</CodeGroup>

<Warning>
  The person who creates the apartment becomes the **Coordinator (koordinatzailea)** and has administrative privileges.
</Warning>

#### Option B: Join an Existing Apartment

If your roommate already created the apartment:

1. Click **"Batu Pisuera"** (Join Apartment)
2. Enter the **5-digit apartment code** your coordinator shared
3. Click **"Join"**

You'll be added as a regular member and can start using the apartment management features.

```php Join Apartment Logic
// From: app/Http/Controllers/PisoController.php:29
public function join(Request $request)
{
    $validated = $request->validate([
        'kodigoa' => 'required|string|exists:pisua,kodigoa',
    ]);

    $piso = Piso::where('kodigoa', $validated['kodigoa'])->first();

    // Add user to apartment (prevents duplicates)
    $request->user()->pisuak()->syncWithoutDetaching([
        $piso->id => ['mota' => 'normala']
    ]);

    SyncInquilinoToOdoo::dispatch($request->user(), $piso);

    session(['pisua_id' => $piso->id]);
    return redirect()->route('dashboard');
}
```

### Step 4: Add Your First Expense

Now let's track a shared expense:

1. Navigate to **"Gastuak"** (Expenses) from the menu
2. Click **"Gastu Berria"** (New Expense)
3. Fill in the expense details:
   - **Izenburua** (Title): What was purchased (max 50 characters)
   - **Zenbatekoa** (Amount): Cost in euros
   - **Data** (Date): When the expense occurred
   - **Ordainduta** (Paid by): Select who paid
4. Click **"Sortu"** (Create)

```php Add Expense Backend
// From: app/Http/Controllers/GastoController.php:59
public function addGasto(Request $request)
{
    $datosProcesados = $request->validate([
        'Nombre' => 'required|string|max:255',
        'Cantidad' => 'required|numeric|min:0',
        'Fecha'  => 'required|date',
        'IdUsuario' => 'required|numeric',
    ]);

    $pisuaId = session('pisua_id');
    $datosProcesados['IdPiso'] = $pisuaId;

    // Save locally
    $gasto = Gasto::create($datosProcesados);

    // Sync to Odoo
    SyncGastoToOdoo::dispatch($gasto);

    return redirect('/gastuak');
}
```

```php Gasto Model
// From: app/Models/Gasto.php
class Gasto extends Model
{
    protected $table = 'gastos';
    protected $primaryKey = 'IdGasto';
    
    protected $fillable = [
        'IdGasto',
        'IdUsuario',
        'IdPiso',
        'Cantidad',
        'Nombre',
        'Fecha',
        'odoo_id',
        'synced',
        'sync_error'
    ];

    protected $casts = [
        'Cantidad'  => 'decimal:2',
        'Fecha'     => 'date',
    ];

    public function usuario()
    {
        return $this->belongsTo(User::class, 'IdUsuario');
    }

    public function piso()
    {
        return $this->belongsTo(Piso::class, 'IdPiso');
    }
}
```

### Step 5: Explore Key Features

Now that you've completed the basics, explore these features:

#### Expense Management
- **View expenses by month**: Filter expenses using the date selector
- **Edit expenses**: Update any expense details
- **Delete expenses**: Remove incorrect entries
- **Settle debts**: Mark payments between roommates

#### Task Management  
- **Create tasks**: Set up recurring household chores
- **Calendar view**: See all tasks in a visual calendar
- **Task states**: Track tasks as "Egiteko" (To-Do), "Egiten" (In Progress), or "Eginda" (Done)
- **Assign tasks**: Distribute responsibilities among roommates

#### Apartment Settings
- **View members**: See all roommates in your apartment
- **Edit apartment**: Change apartment name or code (coordinator only)
- **Manage roles**: Promote members to coordinator (coordinator only)
- **Leave apartment**: Exit an apartment when needed

</Steps>
