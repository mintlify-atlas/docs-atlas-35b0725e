## What is 7POINTS.NET?

7POINTS.NET (also known as "Pisukide") is a comprehensive web application designed specifically for university students living in shared apartments. Born from the 7POINTS team at Migel Altuna LHII, this platform addresses the daily challenges that roommates face when managing household tasks, expenses, and coordination.

### The Problem We Solve

Living with roommates often leads to confusion and conflicts around:
- **Expense Management** - Who paid for what? Who owes whom?
- **Task Distribution** - Who's responsible for cleaning this week?
- **Communication** - How do we stay coordinated without constant messages?
- **Transparency** - How can we ensure fair contribution from everyone?

7POINTS.NET provides a unified solution that brings clarity, fairness, and ease to shared apartment management.

## How It Works

<Steps>
  <Step title="Create or Join an Apartment">
    Start by creating a new apartment or joining an existing one using a unique apartment code shared by your roommates.
  </Step>
  <Step title="Add Roommates">
    Invite your flatmates to join using the apartment code. Assign coordinator roles to trusted members.
  </Step>
  <Step title="Track Expenses">
    Log shared expenses like groceries, utilities, and rent. The system automatically calculates who owes what.
  </Step>
  <Step title="Manage Tasks">
    Create household tasks, assign them to roommates, and track completion with the calendar view.
  </Step>
  <Step title="Stay Coordinated">
    Everyone has real-time visibility into expenses and tasks, eliminating confusion and conflicts.
  </Step>
</Steps>

## Technology Stack

7POINTS.NET is built with modern, reliable technologies:

- **Frontend**: React with TypeScript, Tailwind CSS, and Inertia.js
- **Backend**: Laravel (PHP 8.2) with Fortify for authentication
- **ERP Integration**: Odoo 17.0 for financial management and data centralization
- **Database**: PostgreSQL for robust data storage
- **Deployment**: Docker containers for consistent environments

<Note>
The platform supports multiple languages, including Basque and Spanish, making it accessible to students across different regions.
</Note>

## Built for Students

This platform was developed as a student project at Migel Altuna LHII by the 7POINTS team (Igor Viyuela, Iker Eguizabal, and Luken Franco). It's designed by students, for students, addressing real-world problems faced in shared living situations.

### Key Design Principles

- **Accessibility**: WCAG 2.1 AA compliant with EqualWeb integration for inclusive access
- **Usability**: Clean, intuitive interface with color-coded visual feedback
- **Transparency**: All expenses and tasks visible to all apartment members
- **Fairness**: Automatic debt calculation ensures equitable cost sharing
